## ✦ 人物・背景 (Background/Lore)

モンドの西風騎士団に所属する騎兵隊長。カーンルイアの血を引く謎多き人物。

表向きは陽気で飄々とした性格だが、その内面には複雑な過去と秘密を抱えている。
幼少期にクレープス・ラグヴィンド（ディルックの父）に拾われ、ラグヴィンド家で育てられた。
ディルックとは義兄弟の関係にあるが、現在は複雑な関係性を持つ。

優れた戦闘能力と洞察力を持ち、騎士団の中でも重要な役割を担っている。
一方で、秘密主義的な一面もあり、その真意を読み取ることは難しい。

## ✦ 性能・スキル詳細 (Kit Breakdown)

**概要:**
優秀な氷元素付着能力を持つサブアタッカー。
元素爆発は継続時間が長く、オフフィールドでも氷元素ダメージを与え続けることができる。
凍結編成や溶解編成で活躍する万能なサポーター。

### ❄️ 元素スキル: 「フロストグニル」 (Frostgnaw)

**モーション:** 前方に氷元素を放出して敵にダメージを与える。

- **効果:**
  - 氷元素ダメージを与える
  - 短いクールタイムで連続使用可能
  - クールタイム: 6秒
  - 元素量: 2U（強い氷元素付着）

- **特徴:**
  - シンプルで使いやすい
  - 元素反応を起こしやすい
  - クールタイムが短く、エネルギー回収に優れる

### ❄️ 元素爆発: 「凛冽なる輪舞」 (Glacial Waltz)

**モーション:** 自身の周囲に3つの氷柱を展開し、回転させながら敵を攻撃する。

- **効果:**
  - 出場キャラクターの周囲を氷柱が回転
  - 接触した敵に継続的に氷元素ダメージ
  - 継続時間: 8秒
  - クールタイム: 15秒
  - 元素エネルギー: 60

- **重要な仕様:**
  - オフフィールドでも効果が持続
  - 氷柱は出場キャラクターに追従する
  - 継続的な氷元素付着が可能
  - 凍結編成で特に有効

### ⚔️ 通常攻撃・重撃 (Normal/Charged Attack)

- **通常攻撃:** 最大5段の連続斬り
- **重撃:** スタミナを消費して2段の突進斬り
- **落下攻撃:** 空中から地面に叩きつける

### 固有天賦 (Passive Talents)

1. **冷血の剣:**
   - ガイアの通常攻撃と重撃が氷元素の影響を受けている敵に対して会心率+15%

2. **氷淵の心:**
   - 敵を倒した時、ガイアのHPを15%回復

3. **隠された実力 (探索天賦):**
   - ダッシュのスタミナ消費-20%

## ✦ 天賦育成優先度 (Talent Priority)

**優先度:** **元素爆発 (Q) > 元素スキル (E) > 通常攻撃 (NA)**

- **元素爆発 (Q):** 最優先
  - 主要なダメージソースとなるため
  - オフフィールド運用の要
  - 目標: Lv.8-9

- **元素スキル (E):** 次に優先
  - クールタイムが短く、使用頻度が高い
  - エネルギー回収に重要
  - 目標: Lv.6-8

- **通常攻撃 (NA):** 優先度低
  - サブアタッカー運用では不要
  - メインアタッカー運用でも優先度は低い

## ✦ 命ノ星座 (Constellation)

- **1凸「優れた血統」:**
  - 通常攻撃と重撃が氷元素の影響を受けている敵に対して会心率+15%
  - さらに、敵が凍結状態の場合、会心率がさらに+15%

- **2凸「無限の霜凍」:**
  - 元素爆発「凛冽なる輪舞」の継続時間+2.5秒
  - **重要:** 継続時間が10.5秒になり、元素爆発の回転率が大幅に向上

- **3凸:**
  - 元素スキルのレベル+3

- **4凸「極寒の吻」:**
  - ガイアのHP50%以下の時、シールドを生成（HP上限の30%、継続20秒、クールタイム60秒）
  - シールドは氷元素ダメージに対して250%の吸収効果

- **5凸:**
  - 元素爆発のレベル+3

- **6凸「輪舞の円」:**
  - 元素爆発「凛冽なる輪舞」が追加で1本の氷柱を生成（合計4本）
  - **非常に強力:** ダメージと氷元素付着能力が大幅に向上

**推奨凸数:** 無凸でも十分強力。2凸・6凸が特に有用。

## ✦ チーム編成例 (Team Comps)

### 1. 凍結編成 (Freeze)

ガイアの最も得意な編成。氷風4セットと組み合わせて高火力を発揮。

- **編成:** ガイア / 水元素アタッカー（モナ・タルタリヤ・神里綾人など） / 氷サポーター（ディオナ・ロサリア） / 風サポーター（万葉・スクロース）
- **役割分担:**
  - ガイア: サブアタッカー / 氷元素付着
  - 水アタッカー: 水元素付着
  - 氷サポーター: 氷共鳴 + バフ
  - 風サポーター: 翠緑デバフ + 元素熟知バフ

### 2. 溶解編成 (Melt)

炎元素キャラと組んで溶解反応を狙う。

- **編成:** ガイア / ベネット / 香菱 / 風サポーター（スクロース・万葉）
- **役割分担:**
  - ガイア: サブアタッカー / 溶解アタッカー
  - ベネット: バッファー + ヒーラー
  - 香菱: 炎元素付着 + サブアタッカー
  - 風: 翠緑デバフ

### 3. 物理アタッカー編成

通常攻撃メインのガイアを運用する編成。

- **編成:** ガイア / 雷電将軍（フィッシュル） / ロサリア / ベネット
- **役割分担:**
  - ガイア: 物理メインアタッカー

### 4. 逆溶解サポート編成

炎元素アタッカーの逆溶解をサポート。

- **編成:** ロサリア / ガイア / ベネット / 胡桃（ディルック）

## ✦ 育成のポイント / よくある質問 (Tips & FAQ)

### 育成のポイント

1. **凍結編成が最も強力:**
   - 氷風4セットで会心率を大幅に確保できる
   - 会心ダメージを重視した聖遺物厳選が効率的

2. **元素チャージ効率:**
   - 120-140%を目標にする
   - 元素爆発の継続時間は8秒、クールタイムは15秒なので高回転で回せる

3. **会心バランス:**
   - 凍結編成: 会心率30-40%、会心ダメージ150%以上
   - その他編成: 会心率50-60%、会心ダメージ120%以上

4. **武器選択:**
   - 無課金: 黎明の神剣（★3）が優秀
   - 微課金: 西風剣・天目影打
   - 課金: 磐岩結緑・霧切の廻光

### よくある質問

**Q1: ガイアは強いですか？**
A: ★4キャラクターの中ではトップクラスの性能。特に凍結編成では★5キャラに匹敵する活躍が可能。無凸でも十分強力で、6凸すればさらに強化される。

**Q2: 無凸で使えますか？**
A: 非常に使える。むしろ無凸でも完成度が高い。2凸と6凸が強力だが、無凸でも主力として十分運用可能。

**Q3: メインアタッカーとして使えますか？**
A: 可能だが、サブアタッカー/サポーター運用の方が効率的。物理アタッカーとしての運用も可能だが、元素爆発を活かした運用の方が強力。

**Q4: 黎明の神剣（★3武器）は本当に強いですか？**
A: 凍結編成では非常に強力。HP90%以上で会心率+14%（R5）は★4武器に匹敵する性能。ただし、HPを維持する必要があるため、シールドやヒーラーと組み合わせるのが理想。

**Q5: おすすめの聖遺物周回場所は？**
A: 凍結編成なら「氷風を彷徨う勇士」（フィンドニールの頂上）、サポート特化なら「旧貴族のしつけ」（華池岩岫）を周回。

**Q6: 元素スキルと元素爆発、どちらを先に使うべき？**
A: 元素スキルでエネルギーを回収してから元素爆発を使うのが基本。元素爆発のクールタイムが短いため、高回転で回すことを意識する。

## まとめ

ガイアは**万能な氷元素サブアタッカー・サポーター**です。

### 強み

- 元素爆発の継続時間が長く、オフフィールド性能が優秀
- 凍結編成で特に強力（氷風4セットとのシナジー）
- クールタイムが短く、元素エネルギー回収が容易
- 無凸でも完成度が高い
- ★3武器（黎明の神剣）でも十分な火力を発揮
- 溶解・凍結・超電導など多様な元素反応に対応

### 注意点

- メインアタッカーとしては他キャラに劣る
- 単体での火力は控えめ（チーム編成が重要）
- 元素爆発の範囲が出場キャラクターに追従するため、敵との距離感に注意

### 推奨プレイヤー

- 凍結編成を組みたい人
- 無課金・微課金プレイヤー（コスパが非常に良い）
- オフフィールドサポーターを求めている人
- 序盤から終盤まで長く使えるキャラを探している人

ガイアは初期実装キャラクターながら、現在でも第一線で活躍できる優秀なキャラクターです。
特に凍結編成では★5キャラに匹敵する性能を発揮するため、育成して損はありません。
