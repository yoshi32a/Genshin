## ⚔️ 推奨パーティ編成

コロンビーナの「広範囲継続水撒き」能力を活かした編成を推奨します。

### 【案1】 現実的な「月感電」入門編成

手持ちのキャラで、コロンビーナの「月感電」特性を擬似的に活かす構成です。

| 枠          | 推奨キャラ            | 役割・理由                                                               |
| :---------- | :-------------------- | :----------------------------------------------------------------------- |
| **Main**    | **コロンビーナ (水)** | 月感電の起点。通常攻撃やスキルで水を撒き、感電を会心化させる。           |
| **Sub**     | **フィッシュル (雷)** | **必須枠。** オズによる高頻度の雷付着で、コロンビーナの反応をサポート。  |
| **Sub**     | **リサ (雷)**         | フィッシュルだけでは雷が足りない場合の補助。爆発で防御ダウンも可能。     |
| **Support** | **スクロース (風)**   | 翠緑セットでの耐性ダウンと、**熟知バフ**が月反応のダメージを底上げする。 |

- **戦い方:** フィッシュル/リサで雷設置 → スクロースで拡散 → コロンビーナで殴りつつ月感電を起こす。

### 【案2】 コクセイ主軸・激化パーティ

コクセイをメインアタッカーとして使いたい場合。

| 枠          | 推奨キャラ            | 役割・理由                                                                                  |
| :---------- | :-------------------- | :------------------------------------------------------------------------------------------ |
| **Main**    | **コクセイ (雷)**     | メインアタッカー。激化反応で火力を出す。                                                    |
| **Sub**     | **主人公 (草)**       | 草付着必須。                                                                                |
| **Sub**     | **フィッシュル (雷)** | 雷共鳴＆粒子生成＆追撃。                                                                    |
| **Sustain** | **コロンビーナ (水)** | ごく少量の水付与で「開花」も絡める「超激開花（Quickbloom）」運用、または回復/バフ役として。 |

## 🛠️ 重点育成キャラクター詳細

### 1. 💧 コロンビーナ (Lv.70)

- **現状:** ユーザー内最高レベル。即戦力。
- **武器:** 流浪楽章 (★4) - 火力特化なら最適。なければ鍛造武器など。
- **聖遺物:** 学者/亡命者/教官などの「チャージ効率」や「熟知」が上がるものセット。メインステはHP%や水バフを意識。

### 2. 🌿 主人公 (草) (Lv.50)

- **優先度:** 高。スメール探索＆反応起点として超優秀。
- **武器:** 祭礼の剣 / 西風剣 (チャージ効率重視)。
- **運用:** 元素爆発を回すことが全てです。チャージ効率を最優先に盛ってください。

### 3. ⚡ フィッシュル (Lv.60)

- **優先度:** 中～高。
- **武器:** 絶弦 / 西風猟弓。
- **運用:** オズを出して引っ込むだけですが、それが最強です。

### 4. ⚡ コクセイ (Lv.60)

- **優先度:** 中。
- **武器:** 黎明の神剣 (R5は強力ですが、HP90%維持が必要)。
- **課題:** HP維持のためのシールド役（ノエルなど）か、回避スキルの向上が必要。
