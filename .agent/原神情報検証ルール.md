# 原神情報作成・検証ルール

## 基本原則

1. **推測禁止**: 固有名詞や数値は推測せず、必ず公式情報源で確認する
2. **複数ソース確認**: 可能な限り2つ以上の信頼できる情報源で確認する
3. **最新情報の確認**: バージョン更新で変更される可能性があるため、最新情報を確認する

## 検証が必要な項目

### 1. 固有名詞（絶対に推測禁止）

- [ ] 世界任務名
- [ ] NPC名
- [ ] アイテム名（武器、素材、魚など）
- [ ] 場所名（地域、スポット名）
- [ ] システムの正式名称

### 2. 数値データ（必ず確認）

- [ ] 必要素材の数量
- [ ] ドロップ確率
- [ ] 交換レート
- [ ] リポップ時間
- [ ] 武器ステータス（攻撃力、サブステ数値）
- [ ] 武器効果の数値

### 3. 条件・前提（必ず確認）

- [ ] 解放条件（冒険ランク、前提任務など）
- [ ] 入手方法の正確な手順
- [ ] 使用する餌や道具の正確な名称

### 4. おすすめ情報（根拠が必要）

- [ ] おすすめキャラクター
- [ ] 優先度の根拠
- [ ] 効率的な方法

## 信頼できる情報源

1. 公式Wiki（Genshin Impact Wiki）
2. GameWith、Altema、Kamigameなどの大手攻略サイト
3. 公式フォーラム（HoYoLAB）
4. 複数のプレイヤーによる検証情報

## エラー発見時の対応

1. 即座に修正をコミット
2. 修正内容を明確に記録
3. 同様の誤りが他にないか全体を再確認
