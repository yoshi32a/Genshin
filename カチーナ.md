## ✦ 人物・背景 (Background/Lore)

カチーナはナタ地方の若き戦士で、岩元素の力を操る長柄武器使い。
バージョン5.0の魔神任務で無料配布されるキャラクターで、その後恒常祈願に追加されます。

彼女は「ぐるぐるコマちゃん」という岩元素の構造物を召喚して戦う独特な戦闘スタイルを持ち、
防御力依存の攻撃でダメージを与えます。

---

## ✦ 性能・スキル詳細 (Kit Breakdown)

**概要:**
カチーナは**防御力依存の岩元素サブアタッカー/サポーター**です。
元素スキルで召喚する「ぐるぐるコマちゃん」が継続的に岩元素ダメージを与え、
聖遺物「灰燼の都に立つ英雄の絵巻」4セット効果と組み合わせることで、
**チーム全体に強力なダメージバフ（+40%）を提供**できます。

### 🪨 元素スキル: 「ぐるぐるコマちゃん、いっくよ！」

**モーション:** 岩元素の「ぐるぐるコマちゃん」を召喚する。

- **効果:**
  - フィールドに「ぐるぐるコマちゃん」を配置し、継続的に岩元素範囲ダメージを与える
  - **スキル長押し:** カチーナ自身がコマに乗って移動しつつ攻撃できる
  - ダメージは**カチーナの防御力に依存**
  - 継続時間: 約10-12秒
  - クールタイム: 約20秒

- **重要な仕様:**
  - ナタ地方で「夜魂伝導」という特別な能力を使用可能
  - コマは岩元素創造物として機能

### ⚔️ 元素爆発: 「今こそ本気を出す時！」

**モーション:** 広範囲に岩元素ダメージを与える。

- **効果:**
  - 発動時に周囲の敵に大ダメージ
  - 「ぐるぐるコマちゃん」の能力を強化
  - 元素エネルギー: 70
  - クールタイム: 18秒

### 固有天賦 (Passive Talents)

1. **夜魂の加護:**
   - ナタ地方で夜魂ポイントを消費して特殊移動が可能
   - 探索面でも有用

2. **岩元素共鳴強化:**
   - チーム内の岩元素キャラクターをサポート

3. **探索天賦:**
   - ナタ地方での探索に役立つ能力

## ✦ 天賦育成優先度 (Talent Priority)

### サポート型運用（推奨）

**優先度:** **育成不要（全て Lv.1 でOK）**

- **理由:** 「灰燼の都に立つ英雄の絵巻」4セット効果は天賦レベルに依存しない
- カチーナの最大の強みは**40%ダメージバフの提供**であり、これは聖遺物効果
- 天賦レベルを上げても聖遺物バフには影響しないため、育成コストを大幅に削減可能

### サブアタッカー型運用（火力重視の場合）

**優先度:** **元素スキル (E) ≧ 元素爆発 (Q) \u003e 通常攻撃 (NA)**

- **元素スキル (E):** 最優先
  - コマのダメージが上がる
  - 目標: Lv.6-8

- **元素爆発 (Q):** 次に優先
  - 爆発ダメージが上がる
  - 目標: Lv.6-8

- **通常攻撃 (NA):** 最低優先
  - ほとんど使わないため、Lv.1で十分

## ✦ チーム編成例 (Team Comps)

### 1. 岩サポート編成（推奨）

チーム内の岩元素キャラクターに+40%ダメージバフを提供する編成。

- **編成:** 岩元素メインアタッカー / カチーナ / 他の岩元素キャラ / フレックス枠
- **例:** ノエル (6凸) / カチーナ / ゴロー / 鍾離
- **役割分担:**
  - カチーナ: 結晶反応で灰燼バフを発動 + 岩粒子供給
  - ノエル: メインアタッカー（バフを受ける）
  - ゴロー: 防御力バフ
  - 鍾離: シールド + 耐性ダウン

**評価:** 岩元素アタッカーの火力を最大化できる。

### 2. 結晶反応パーティ

炎元素など、結晶反応を起こしやすい元素のキャラと組む。

- **編成:** カチーナ / 岩元素アタッカー / 炎/雷/氷/水元素キャラ / ヒーラー
- **例:** カチーナ / アルベド / ベネット / 香菱
- **役割分担:**
  - カチーナ: 結晶反応で灰燼バフを維持
  - アルベド: オフフィールド火力
  - ベネット/香菱: 炎共鳴 + 結晶反応トリガー

**評価:** 灰燼バフを継続的に発動しやすい。

### 3. 岩共鳴編成

岩元素共鳴でシールド強化とダメージバフを得る。

- **編成:** カチーナ / 岩元素メインアタッカー / 岩元素サポート / フレックス枠
- **役割分担:**
  - 岩元素共鳴: シールド強化+15%、岩耐性-20%、ダメージ+15%

**評価:** 岩元素特化の安定編成。

## ✦ 育成のポイント / よくある質問 (Tips & FAQ)

### 育成のポイント

1. **サポート型が最もコスパ良好:**
   - 天賦レベルを上げる必要がないため、育成コストが極めて低い
   - 「灰燼の都に立つ英雄の絵巻」4セットを装備するだけで役割を果たせる
   - 聖遺物のレベルすら上げなくて良いという声も（ただしHP確保は推奨）

2. **結晶反応を維持する:**
   - 灰燼バフを発動・維持するには結晶反応が必須
   - 炎・雷・氷・水元素のキャラと組むことで結晶反応を継続的に発生させる

3. **元素チャージ効率の確保:**
   - 元素エネルギー70は重め
   - 元素チャージ効率を130-150%確保するか、西風長槍で粒子生成

4. **岩元素アタッカーと組む:**
   - カチーナの最大の強みは岩元素キャラへの+40%ダメージバフ
   - ノエル、アルベド、凝光、荒瀧一斗などの岩アタッカーと組むのが最適

5. **無料配布キャラの利点:**
   - Ver 5.0の魔神任務クリアで無料入手可能
   - その後恒常祈願に追加されるため、凸も進めやすい

### よくある質問

**Q1: カチーナは初心者におすすめですか？**
A: はい。無料配布されるうえ、サポート運用なら天賦育成不要で即戦力になります。岩元素アタッカーを持っている、またはノエルを育成中なら特におすすめ。

**Q2: 天賦は育成しなくていいって本当？**
A: 本当です（サポート運用の場合）。灰燼絵巻4セットの+40%ダメージバフは聖遺物効果であり、カチーナ自身の天賦レベルや能力値に依存しません。そのため、天賦を全てLv.1のままでも問題なく機能します。

**Q3: 聖遺物のレベルも上げなくていいって本当？**
A: 理論上は正しいですが、事故（カチーナがやられる）を防ぐためにHPを上げておくことを推奨する意見もあります。最低限のHP確保は考慮しましょう。

**Q4: 無凸でも使えますか？**
A: 使えます。サポート運用なら無凸で十分です。サブアタッカー運用なら2凸以上あると良いですが、必須ではありません。

**Q5: 灰燼の都に立つ英雄の絵巻はどこで集めるべき？**
A: ナタ地方の秘境で入手可能です。Ver 5.0以降で解放されます。

**Q6: どのキャラクターと相性が良いですか？**
A: 岩元素アタッカー全般と相性抜群です。特にノエル（6凸）、荒瀧一斗、凝光、アルベドなど。また、結晶反応を起こせる炎・雷・氷・水元素キャラとも相性が良いです。

**Q7: サブアタッカー運用とサポート運用、どちらが良い？**
A: 初心者や低リソース環境ではサポート運用を強く推奨します。育成コストが極めて低く、即戦力になるためです。十分なリソースがあり、火力も出したい場合のみサブアタッカー運用を検討しましょう。

## まとめ

カチーナは**無料配布の岩元素サポーター**で、育成コストが極めて低いのが最大の特徴です。

### 強み

- **圧倒的な育成コストの低さ**（サポート運用なら天賦Lv.1でOK）
- 岩元素キャラに+40%ダメージバフを提供（灰燼絵巻4セット）
- 無料配布で確実に入手可能
- 恒常祈願追加で凸も進めやすい
- 防御力依存で生存力が高い
- ナタ地方での探索に便利

### 注意点

- サポート運用には「灰燼の都に立つ英雄の絵巻」4セットが必須
- 結晶反応を維持する必要がある（他の元素キャラが必要）
- 岩元素アタッカーがいないと真価を発揮できない
- サブアタッカー運用は育成コストが高い
- 元素エネルギー70は重め（元素チャージ管理が必要）

### 推奨プレイヤー

- 岩元素アタッカーを使っている人
- ノエル（6凸）を使っている人
- 育成リソースが限られている初心者
- 低コストで即戦力のサポーターが欲しい人
- ナタ地方の探索を快適にしたい人

カチーナは「灰燼の都に立つ英雄の絵巻」4セットと組み合わせることで、
**最小限の育成コストで最大限のサポート性能を発揮する**優秀なキャラクターです。
特に岩元素アタッカーを使っているプレイヤーには必須級のサポーターと言えるでしょう。
