# 原神キャラクター管理システム

原神（Genshin Impact）のキャラクター情報を管理・可視化するWebアプリケーションです。

## 🎮 概要

所持キャラクターの情報を一元管理し、育成の優先度やレベル、天賦などをビジュアルで確認できるシングルページアプリケーションです。

## ✨ 機能

- **キャラクター一覧表示**: 所持キャラクター17体の詳細情報を表示
- **フィルタリング機能**:
  - 属性別フィルター（炎・水・風・雷・草・氷・岩）
  - 育成優先度フィルター（最優先・育成推奨・保留）
- **ソート機能**:
  - レベル順
  - レアリティ順（星4・星5）
  - 優先度順
- **統計情報**:
  - 所持キャラクター数
  - 平均レベル
  - 最高レベル
  - 優先育成キャラ数
- **レスポンシブデザイン**: PC・スマートフォンに対応
- **キャラクター詳細情報**: 各キャラクターのマークダウンファイルで詳細管理

## 🚀 使い方

### ローカルで実行

1. リポジトリをクローン:
```bash
git clone <repository-url>
cd Genshin
```

2. `index.html` をブラウザで開く:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

または、シンプルなHTTPサーバーを起動:
```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsの場合
npx serve
```

ブラウザで `http://localhost:8000` にアクセス

## 🛠 技術スタック

- **フロントエンド**: React 18 (CDN版)
- **スタイリング**:
  - Tailwind CSS
  - カスタムグラデーション（属性・レアリティ別）
- **言語**: JavaScript (Babel standalone)
- **フォント**: Google Fonts (Noto Sans JP)

## 📊 管理情報

### プレイヤー情報
- 冒険者ランク: 40
- 魔神任務: 3-1
- 所持キャラクター: 17体
- 最高レベルキャラ: Lv.70

### キャラクター管理
各キャラクターについて以下を管理:
- レベル
- 凸数（命ノ星座）
- 天賦レベル
- 装備武器
- 育成優先度

## 🎨 デザイン特徴

- **属性カラー**: 各属性に応じたグラデーション配色
- **育成優先度**: 視覚的に識別しやすい色分け（赤・黄・緑）
- **レスポンシブ**: モバイル・タブレット・デスクトップに最適化
- **ダークテーマ**: 目に優しいダークモードデザイン

## 🔧 カスタマイズ

キャラクター情報を更新する場合は、`index.html` 内の `characters` 配列を編集してください:

```javascript
const characters = [
    {
        name: "キャラ名",
        element: "属性",
        rarity: 5,
        level: 70,
        constellation: 0,
        talent: "1/1/1",
        weapon: "武器名",
        priority: "high"
    },
    // ...
];
```
